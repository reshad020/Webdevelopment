Module:2 Basic CSS
  1. Using percentage(%) parameters in css can make my website responsive.
  2. Use for styling small portion of text.
  3. To style a border : (border : size type color;) size = dotted,solid,dashed...etc
  4. Margin defines space around an HTML element. To style a margin : (margin: top right bottom left;)
  5. Padding is used to create space around an element’s content , inside of any defined borders. (padding: top right bottom left;)
  6. (text-align: right | left | center | justify | inherit | initial;) *Justify will align the text from both left and right side.
  7. A block level element always starts on a new line and takes the full width available. To make the element inline, we can use(display:inline;)
  8. An inline element does not start on a new line and only takes as much as width necessary. To make the element block, we can use(display:block;)
  9. Create external css file = < head > < link rel=”stylesheet” href=”style.css”/ >
